
# GitHub Badges Project

Bu proje, çeşitli GitHub rozetlerini ve başarılarını kazanmak için oluşturulmuştur.

## Hedeflenen Rozetler
- **YOLO**: Kod incelemesi olmadan PR birleştirme.
- **Quickdraw**: Bir issue veya PR'ı açıldıktan sonra 5 dakika içinde kapatma.
- **Pull Shark**: PR birleştirme.
- **Pair Extraordinaire**: `Co-authored-by` ile commit atma.

## Nasıl Kullanılır?
Bu repoda yapılan işlemler, GitHub profilinize aktivite ve başarı olarak yansıyacaktır.
